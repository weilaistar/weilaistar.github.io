How to create personal blog?

2025-11-10

如何用 GitHub 和 Hexo 搭建你的第一个个人博客？

这是我的第一篇博客，希望能帮助所有像我一样，渴望拥有一个纯净、免费且完全掌控的个人主页的朋友。

我们将使用强大的静态博客生成器 Hexo 和免费的代码托管平台 GitHub Pages，在极短的时间内搭建起一个优雅的个人博客。

一、 核心技术选型与优势

Hexo + GitHub Pages 是极客们常用的博客组合，其优势在于：

  1. 纯粹且快速 (Hexo)： 基于 Node.js 的静态博客框架，不依赖数据库，速度极快，专注于 Markdown 写作。
  2. 免费且稳定 (GitHub Pages)： 提供免费的静态网站托管，高稳定性，天然支持 Git 版本控制。
  3. 高度自定义： 拥有庞大的主题和插件生态系统，可深度定制。

二、 本地环境准备

在本地电脑上，我们需要安装以下基础环境：

软件 作用 检查命令
Node.js Hexo 运行环境（推荐 LTS 版本） node -v
Git 版本控制和部署工具 git --version

安装完成后，全局安装 Hexo 命令行工具：

npm install -g hexo-cli

## 三、 Hexo 初始化与本地运行

在确保本地已安装 **Node.js**、**Git** 和 **Hexo CLI** 的前提下，可以开始搭建博客骨架。

### 1. 创建并初始化项目

为你未来的博客选择一个存放目录，然后执行初始化命令，`<project-folder-name>` 是你的项目文件夹名称：

```bash
# 切换到你的目标目录
cd /path/to/your/workspace

# 初始化博客项目
hexo init <project-folder-name>
cd <project-folder-name>

# 安装项目所需的依赖（Hexo核心组件）
npm install
